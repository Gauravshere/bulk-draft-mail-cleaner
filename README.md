# Bulk Draft Mail Cleaner

An advanced WordPress admin plugin to bulk delete draft emails/posts.

## Features
- Lists all draft posts
- Bulk select and delete
- Secure AJAX handling
- Admin-only access
- Clean UI

## Use Case
Useful for cleaning up large numbers of draft emails or posts created during testing.

## Tech Stack
- PHP
- WordPress
- AJAX
- jQuery

## Installation
1. Copy plugin to wp-content/plugins/
2. Activate from admin
3. Navigate to "Draft Mail Cleaner"
